# Detectra Frontend
Frontend interface for Detectra - Smart Phishing Detection System.

## Overview
Detectra is a phishing detection platform designed to help users identify potentially malicious URLs and websites. The frontend provides an intuitive interface for submitting URLs and viewing phishing detection results.

## Technologies Used
* React.js
* TypeScript
* Vite
* Tailwind CSS
* JavaScript
* HTML5
* CSS3

## Features
* User-friendly interface
* URL submission and analysis
* Responsive design
* Real-time result display
* Integration with backend phishing detection services

## Project Structure
* `src/` - Application source code
* `public/` - Static assets
* `components/` - Reusable UI components
* `pages/` - Application pages
